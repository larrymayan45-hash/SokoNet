# 🚀 SokoNet

SokoNet is a smart digital marketplace designed to connect businesses, service providers, and customers in one powerful platform. It enables users to buy and sell products, find jobs, and access services efficiently.

---

## 🌍 Vision

To become Africa’s leading digital ecosystem for commerce and job connectivity.

---

## 🎯 Features

* 🛒 **Marketplace System** – Buy and sell products (food, goods, etc.)
* 👷 **Service Marketplace** – Connect with workers (plumbers, electricians, etc.)
* 🔐 **Authentication System** – Secure login with OTP verification
* 📱 **Mobile-First Design** – Optimized for accessibility and usability
* 💬 **Real-time Communication** *(Planned)*
* 💳 **Payment Integration (M-Pesa)** *(Planned)*
* ⭐ **Ratings & Reviews System**
* 📊 **User Dashboard**
* 🔎 **Search & Filter System**
* 📦 **Order Management**
* 👤 **User Profiles**
* 🧠 **Smart Recommendations** *(Future AI integration)*

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/SokoNet.git
```

2. Navigate to the project:

```
cd SokoNet/mobile
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file and add:

```
REACT_APP_API_URL=your_backend_url
```
